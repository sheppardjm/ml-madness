Create `src/ui/advancement_table.py` with:

**Function: `build_advancement_df(mc_result, team_id_to_name, team_id_to_seed, all_team_ids) -> pd.DataFrame`**

Parameters:
- `mc_result`: Dict from `simulate_bracket(mode="monte_carlo")` -- has `advancement_probs` dict
- `team_id_to_name`: Dict mapping team_id -> display name
- `team_id_to_seed`: Dict mapping team_id -> seed label (e.g., "W01")
- `all_team_ids`: List of all 68 team IDs in the tournament (from seedings.values())

Returns: pandas DataFrame with columns:
- "Team" (str): Team display name
- "Seed" (str): Seed label (e.g., "W01", "X16a")
- "SeedNum" (int): Integer seed number (for sorting)
- "Round of 64" (float): P(team wins at least one R1 game) -- 0.0 to 1.0
- "Round of 32" (float): P(team reaches Round of 32) -- 0.0 to 1.0
- "Sweet 16" (float): P(team reaches Sweet 16)
- "Elite 8" (float): P(team reaches Elite 8)
- "Final Four" (float): P(team reaches Final Four)
- "Championship" (float): P(team reaches Championship game)
- "Champion" (float): P(team wins tournament)

Logic:
1. Define `ROUND_COLS = ["Round of 64", "Round of 32", "Sweet 16", "Elite 8", "Final Four", "Championship", "Champion"]`
2. For each team_id in `all_team_ids`:
   - Look up `probs = mc_result["advancement_probs"].get(team_id, {})`
   - Build a row dict with team name, seed, seednum, and probability for each round
   - For rounds not in `probs`, use 0.0 (this handles First Four losers per pitfall 4)
3. Create DataFrame from rows
4. Sort by "Champion" descending, then by "SeedNum" ascending as secondary sort
5. Return the DataFrame

IMPORTANT: Use `all_team_ids` (all 68 teams from seedings) as the outer loop, NOT advancement_probs.keys(). This ensures First Four losers who have zero advancement probability still appear in the table. This is the LEFT JOIN pattern from research pitfall 4.

**Function: `get_round_column_config() -> dict`**

Returns a dict suitable for `st.dataframe(column_config=...)` that configures ProgressColumn for each round column:
```python
{
    col: st.column_config.ProgressColumn(
        col,
        format="%.1f%%",
        min_value=0.0,
        max_value=1.0,
    )
    for col in ROUND_COLS
}
```

Note: This function must be called at runtime within a Streamlit context (because st.column_config is only available when Streamlit is running). Do NOT call it at module import time.

Import: `import pandas as pd` and `import streamlit as st` (st only used inside get_round_column_config).

   ```python
   with tab_advancement:
       st.subheader("Round-by-Round Advancement Probabilities")
       st.caption("Based on 10,000 Monte Carlo simulations. Click any column header to sort.")

       all_team_ids = list(seedings.values())
       adv_df = build_advancement_df(mc_result, team_id_to_name, team_id_to_seed, all_team_ids)

       # Configure ProgressColumn for visual probability bars
       column_config = get_round_column_config()

       st.dataframe(
           adv_df,
           column_config=column_config,
           hide_index=True,
           use_container_width=True,
           height=800,  # Show many rows without excessive scrolling
       )

       # Summary stats below the table
       col1, col2, col3 = st.columns(3)
       with col1:
           n_teams_ff = len(adv_df[adv_df["Final Four"] > 0.0])
           st.metric("Teams with Final Four chance", n_teams_ff)
       with col2:
           n_teams_champ = len(adv_df[adv_df["Champion"] > 0.0])
           st.metric("Teams with Champion chance", n_teams_champ)
       with col3:
           top_champ = adv_df.iloc[0]
           st.metric("Top Champion Probability",
                     f"{top_champ['Champion']:.1%}",
                     delta=top_champ["Team"])
   ```
