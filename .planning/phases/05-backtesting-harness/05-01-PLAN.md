**Function `build_actual_slot_winners(season, processed_dir="data/processed", seed_round_slots_csv="data/raw/kaggle/MNCAATourneySeedRoundSlots.csv")`:**
- Uses DuckDB to join `seeds.parquet` (filtered to season) with `MNCAATourneySeedRoundSlots.csv` (NO season column -- universal routing table) to get each team's slot assignments.
- Then joins to `tournament_games.parquet` (filtered to season) to find actual winners per slot.
- CRITICAL: The CSV has NO Season column. Season isolation comes from filtering `seeds.parquet` by season.
- SQL pattern (verified in research):
  ```sql
  WITH team_slots AS (
      SELECT DISTINCT s.TeamID, sr.GameSlot, sr.GameRound,
             sr.EarlyDayNum, sr.LateDayNum
      FROM read_parquet('{seeds_parquet}') s
      JOIN read_csv('{seed_round_slots_csv}') sr ON s.Seed = sr.Seed
      WHERE s.Season = {season}
  )
  SELECT ts.GameSlot AS slot_id, g.WTeamID AS actual_winner
  FROM team_slots ts
  JOIN read_parquet('{games_parquet}') g
      ON g.Season = {season}
      AND g.DayNum BETWEEN ts.EarlyDayNum AND ts.LateDayNum
      AND (g.WTeamID = ts.TeamID OR g.LTeamID = ts.TeamID)
      AND g.WTeamID = ts.TeamID
  ORDER BY ts.GameSlot
  ```
- Returns `dict[str, int]` mapping slot_id -> actual_winner_team_id.
- Verify: Returns exactly 67 slot winners for any post-2010 season.

**Function `score_bracket(predicted_slots, actual_winners)`:**
- `predicted_slots`: dict mapping slot_id -> team_id (from simulate_bracket result).
- `actual_winners`: dict mapping slot_id -> team_id (from build_actual_slot_winners).
- For each slot in actual_winners:
  - Get round number via `slot_round_number(slot_id)` from bracket_schema.
  - Skip round 0 (First Four) -- not scored in ESPN brackets.
  - Get round name via `ROUND_NAMES[round_num]`.
  - Compare `predicted_slots.get(slot_id)` to `actual_winner`.
  - Accumulate ESPN points, per-round correct/total counts.
- Return dict:
  ```python
  {
      'espn_score': int,
      'espn_max': 1920,
      'per_round_accuracy': {round_name: float},
      'per_round_correct': {round_name: int},
      'per_round_total': {round_name: int},
  }
  ```

**Function `compute_game_metrics(test_df, feature_cols, scaler, calibrated_clf)`:**
- Takes the test-year matchup DataFrame and the fold-specific scaler + calibrated model.
- Computes Brier score, log-loss, accuracy, upset detection rate using the same pattern as `evaluate_all_holdout_years()` in evaluate.py.
- Do NOT call predict_fn individually per game. Use batch mode: `scaler.transform(test_df[feature_cols].values)` then `calibrated_clf.predict_proba()`.
- Return dict:
  ```python
  {
      'brier': float,
      'log_loss': float,
      'accuracy': float,
      'n_games': int,
      'n_upsets': int,
      'upset_correct': int,
      'upset_detection_rate': float,
  }
  ```
