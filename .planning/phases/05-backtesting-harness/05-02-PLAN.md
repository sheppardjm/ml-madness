**Imports:**

**Function `backtest(year_range=None, model="baseline", model_path="models/logistic_baseline.joblib", processed_dir="data/processed", seed_round_slots_csv="data/raw/kaggle/MNCAATourneySeedRoundSlots.csv", slots_csv="data/raw/kaggle/MNCAATourneySlots.csv", output_path="backtest/results.json")`:**

Implementation steps:

1. **Validate inputs**: If `year_range` is None, default to `BACKTEST_YEARS` ([2022,2023,2024,2025]). If `model != "baseline"`, raise `ValueError` (only baseline supported for now).

2. **Load best_C from artifact** (NOT the model itself): `artifact = joblib.load(model_path)` then `best_C = artifact["best_C"]`. Print the extracted value. Do NOT use `artifact["model"]` or `artifact["scaler"]` for predictions.

3. **Build shared data structures** (once, outside the year loop):
   - `df = build_matchup_dataset(processed_dir)` -- full matchup dataset across all seasons
   - `stats_lookup = build_stats_lookup(processed_dir)` -- all team stats keyed by (season, team_id). The (year, team_id) key structure ensures no cross-year contamination.

4. **Per-year loop** -- for each `test_year` in `year_range`:

   a. **Re-fit model on pre-year data** (temporal isolation):
      ```python
      train_df = df[df['Season'] < test_year].copy()
      X_train = train_df[FEATURE_COLS].values
      y_train = train_df['label'].values

      scaler = StandardScaler()
      X_train_scaled = scaler.fit_transform(X_train)

      clf = LogisticRegression(
          C=best_C, class_weight='balanced',
          solver='lbfgs', max_iter=1000, random_state=42
      )
      clf.fit(X_train_scaled, y_train)
      calibrated_clf = ClippedCalibrator(clf, clip_lo=CLIP_LO, clip_hi=CLIP_HI)
      ```

   b. **Build year-specific predict_fn closure**:
      - IMPORTANT: Use a default argument `_year=test_year` to avoid closure-over-loop-variable bug.
      - Wrap `compute_features()` in try/except that returns 0.5 for missing teams (First Four teams may be missing from stats_lookup).
      ```python
      def predict_fn(team_a_id: int, team_b_id: int, _year=test_year, _scaler=scaler, _clf=calibrated_clf) -> float:
          try:
              features = compute_features(_year, team_a_id, team_b_id, stats_lookup)
              x = np.array([features[name] for name in FEATURE_COLS]).reshape(1, -1)
              x_scaled = _scaler.transform(x)
              return float(_clf.predict_proba(x_scaled)[0, 1])
          except KeyError:
              return 0.5  # No information for missing teams (First Four edge case)
      ```

   c. **Load seedings and simulate bracket**:
      ```python
      seedings = load_seedings(season=test_year)
      sim_result = simulate_bracket(
          seedings, predict_fn, mode='deterministic',
          season=test_year, stats_lookup=stats_lookup
      )
      predicted_slots = {sid: data['team_id'] for sid, data in sim_result['slots'].items()}
      ```

   d. **Get actual slot winners and compute bracket metrics**:
      ```python
      actual_winners = build_actual_slot_winners(
          test_year, processed_dir, seed_round_slots_csv
      )
      bracket_metrics = score_bracket(predicted_slots, actual_winners)
      ```

   e. **Compute game-level metrics** using fold-specific scaler + model:
      ```python
      test_df = df[df['Season'] == test_year].copy()
      game_metrics = compute_game_metrics(test_df, FEATURE_COLS, scaler, calibrated_clf)
      ```

   f. **Combine into per-year result dict**:
      ```python
      year_result = {
          'year': test_year,
          **game_metrics,      # brier, log_loss, accuracy, n_games, n_upsets, etc.
          **bracket_metrics,   # espn_score, espn_max, per_round_accuracy, etc.
      }
      ```

5. **Compute summary aggregates**:
   ```python
   summary = {
       'mean_brier': float(np.mean([r['brier'] for r in per_year_results])),
       'mean_log_loss': float(np.mean([r['log_loss'] for r in per_year_results])),
       'mean_accuracy': float(np.mean([r['accuracy'] for r in per_year_results])),
       'mean_espn_score': float(np.mean([r['espn_score'] for r in per_year_results])),
   }
   ```

6. **Build final results dict**:
   ```python
   results = {
       'model': model,
       'year_range': year_range,
       'generated_at': str(date.today()),
       'per_year': per_year_results,
       'summary': summary,
   }
   ```

7. **Write to JSON** (create directory if needed):
   ```python
   out_path = pathlib.Path(output_path)
   out_path.parent.mkdir(parents=True, exist_ok=True)
   out_path.write_text(json.dumps(results, indent=2))
   ```

8. **Print formatted comparison table** showing per-year results:
   ```
   === BACKTEST RESULTS (baseline) ===
   Year | Brier  | LogLoss | Accuracy | ESPN Score | Upsets Hit | Upset Rate
   ---- | ------ | ------- | -------- | ---------- | ---------- | ----------
   2022 | 0.XXXX | 0.XXXX  | XX.X%    | XXXX/1920  | X/XX       | XX.X%
   ...
   Mean | 0.XXXX | 0.XXXX  | XX.X%    | XXXX/1920  | -          | -
   ```
   Also print per-round accuracy for each year (compact format).

9. **Return** the results dict.
