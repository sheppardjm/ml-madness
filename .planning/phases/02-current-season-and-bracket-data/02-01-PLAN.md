    3. Create `src/ingest/cbbdata_client.py` with:

       **Constants:**
       ```python
       CBD_BASE_URL = "https://www.cbbdata.com/api"
       ```

       **`get_cbbdata_token(username: str | None = None, password: str | None = None) -> str`**
       - If username/password not provided, read from env vars CBD_USERNAME and CBD_PASSWORD
       - POST to `{CBD_BASE_URL}/auth/login` with JSON body `{"username": ..., "password": ...}`
       - Response is a JSON list; API key is `response.json()[0]`
       - Return the API key string
       - Raise ValueError with clear message if env vars missing and no args provided
       - Use requests.Session with HTTPAdapter(max_retries=Retry(total=3, backoff_factor=1)) for transient failure handling
       - timeout=30 for all requests

       **`fetch_torvik_ratings(api_key: str, year: int = 2026) -> pd.DataFrame`**
       - GET `{CBD_BASE_URL}/torvik/ratings` with params `{"key": api_key, "year": year}`
       - Response is raw Parquet bytes — decode with `pd.read_parquet(BytesIO(response.content))`
       - Return DataFrame (columns include: team, conf, barthag, adj_o, adj_d, adj_t, wab, year, and rank columns)
       - timeout=60 (larger dataset)

       **`fetch_cbbdata_teams() -> pd.DataFrame`**
       - GET `{CBD_BASE_URL}/data/teams` (no API key required)
       - Response is raw Parquet bytes — decode with `pd.read_parquet(BytesIO(response.content))`
       - Return DataFrame with ESPN IDs, slugs, and team info
       - timeout=30

       **IMPORTANT anti-patterns to avoid:**
       - Do NOT use `response.json()` for data endpoints — they return Parquet bytes, not JSON
       - Do NOT store API key in source code — always read from env vars or function params
       - Do NOT use the R-style `cbd_login()` package — use direct HTTP POST

       Add `if __name__ == "__main__"` block that:
       - Calls get_cbbdata_token() (from env vars)
       - Calls fetch_torvik_ratings(key, 2026)

       - Calls fetch_cbbdata_teams()
       - Prints teams DataFrame shape and column names
