**Add helper function `_build_prob_matrix(teams, team_seed_map, predict_fn)`:**
- `teams`: list of all unique kaggle_team_ids (from seedings.values()).
- `team_seed_map`: dict[int, int] mapping kaggle_team_id -> seed_num.
- `predict_fn`: callable(team_a_id, team_b_id) -> float.
- Pre-compute P[i, j] = P(teams[i] beats teams[j]) for all 68x68 pairs.
- **Enforce canonical ordering**: If teams[i] has seed_num <= teams[j] seed_num, call predict_fn(teams[i], teams[j]) directly. Otherwise, call predict_fn(teams[j], teams[i]) and take 1.0 - result.
- For equal teams (i == j), set P[i, j] = 0.5.
- **Handle same-seed matchups**: When two teams have the same seed number (First Four opponents), use a tiebreaker: sort by team_id (lower team_id = team_a) to maintain consistent canonical ordering. This avoids incorrect probability inversion.
- Return the numpy array of shape (n_teams, n_teams) with dtype float64.
- Also return `team_to_idx` (dict: kaggle_team_id -> matrix index) and `idx_to_team` (dict: matrix index -> kaggle_team_id).

**Add helper function `_compute_advancement_probs(occupants, slot_tree, idx_to_team, team_seed_map, n_runs)`:**
- `occupants`: dict[str, np.ndarray] -- slot_id -> array of shape (n_runs,) containing team indices.
- For each team, compute the fraction of simulations where they appear as the winner in each round.
- Group slots by round number (using `slot_round_number()`).
- For round R, a team "advances to round R" if they win any slot in round R. For round 0 (First Four), a team advances if they win a FF slot.
- Return `dict[int, dict[str, float]]` mapping kaggle_team_id -> {round_name: probability}.
  - Use ROUND_NAMES for human-readable round names.
  - Only include rounds where the team has > 0 probability.
  - Include a special "Champion" entry for teams that win R6CH.

**Monte Carlo implementation inside `simulate_bracket()`:**
1. Build team list, team_to_idx, idx_to_team from seedings.
2. Build probability matrix via `_build_prob_matrix()`. Print: `f"Probability matrix: {n_teams}x{n_teams} ({n_teams*n_teams} predict_fn calls)"`.
3. Create numpy RNG: `rng = np.random.default_rng(seed)`.
4. Initialize `occupants: dict[str, np.ndarray]` -- will hold shape (n_runs,) arrays of team indices for each slot.
5. Traverse slots in topological order:
   - For each slot, resolve StrongSeed and WeakSeed:
     - If reference is a seed label (in seedings): `strong_occ = np.full(n_runs, team_to_idx[seedings[ref]], dtype=np.int32)`
     - Else (prior slot): `strong_occ = occupants[ref]`
   - Vectorized Bernoulli draw:
     ```python
     probs = prob_matrix[strong_occ, weak_occ]  # shape (n_runs,)
     draws = rng.random(n_runs)                   # shape (n_runs,)
     occupants[slot_id] = np.where(draws < probs, strong_occ, weak_occ)
     ```
6. Compute champion:
   - `champion_occ = occupants['R6CH']`
   - Use `np.bincount()` to count wins per team index.
   - Champion = team with most wins. Confidence = count / n_runs.
7. Compute advancement probabilities via `_compute_advancement_probs()`.
8. Build the return dict:
   ```python
   result = {
       'mode': 'monte_carlo',
       'season': season,
       'n_runs': n_runs,
       'champion': {
           'team_id': int(idx_to_team[champion_idx]),
           'confidence': float(champion_counts[champion_idx] / n_runs),
       },
       'advancement_probs': advancement_probs,  # team_id -> {round: prob}
   }
   ```
   - **Important**: Convert ALL numpy types to native Python (int, float) before returning. Use `int()` for team IDs and `float()` for probabilities.
