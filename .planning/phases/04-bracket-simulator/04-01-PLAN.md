**Function `build_slot_tree(season=2025, slots_csv=SLOTS_CSV)`:**
- Uses DuckDB to read `MNCAATourneySlots.csv` filtered to the given season.
- Returns a dict with keys:
  - `'slots'`: dict mapping slot_id -> `{'StrongSeed': str, 'WeakSeed': str}` (from DataFrame set_index('Slot').to_dict('index'))
  - `'order'`: list of slot IDs in topological order (FF slots first, then R1, R2, R3, R4, R5, R6)
  - `'ff_slots'`: set of First Four slot IDs (those not starting with 'R')
- Topological ordering: sort by round key where FF slots get key 0, R1 gets 1, R2 gets 2, etc. (use `int(slot_id[1])` for R-prefixed slots).
- Assert the DataFrame has exactly 67 rows for any post-2010 season. Print a clear error if not: `f"Expected 67 slots for season {season}, got {len(df)}"`.

**Function `get_topological_order(slot_tree)`:**
- Convenience wrapper: returns `slot_tree['order']`.

**Function `load_seedings(season=2025, seeds_parquet=SEEDS_PARQUET)`:**
- Uses DuckDB to read seeds.parquet filtered to the given season.
- The `Seed` column contains values like 'W01', 'W16a', 'W16b' -- these are the seed labels.
- Returns `dict[str, int]` mapping seed_label -> kaggle_team_id (TeamID column).
- Assert the result has 68 entries for post-2010 seasons: `f"Expected 68 seedings for season {season}, got {len(result)}"`.

**Function `build_team_seed_map(seedings)`:**
- Takes the seedings dict from `load_seedings()`.
- Returns `dict[int, int]` mapping kaggle_team_id -> integer seed number.
- Parse seed number from the seed label: strip the region letter prefix (first char), strip any trailing 'a'/'b' suffix, convert remainder to int. Examples: 'W01' -> 1, 'X16a' -> 16, 'Y11b' -> 11, 'Z05' -> 5.
- Multiple teams can have the same seed number (First Four teams share the same seed number).

**Function `build_predict_fn(model_path='models/logistic_baseline.joblib', processed_dir='data/processed', season=2025)`:**
- Imports `load_model`, `predict_matchup` from `src.models.train_logistic`.
- Imports `compute_features`, `build_stats_lookup` from `src.models.features`.
- Loads model, scaler, feature_names via `load_model(model_path)`.
- Builds stats_lookup via `build_stats_lookup(processed_dir)`.
- Returns a closure `predict_fn(team_a_id, team_b_id) -> float` where:
  - `team_a_id` must have the lower seed number (canonical ordering -- caller's responsibility).
  - Calls `compute_features(season, team_a_id, team_b_id, stats_lookup)` to get features.
  - Calls `predict_matchup(features, model, scaler, feature_names)` to get probability.
  - Returns the probability (float in (0, 1)).
- Also returns the stats_lookup as a second return value (needed by score_predictor.py in plan 04-04):
  `return predict_fn, stats_lookup`

**Function `slot_round_number(slot_id)`:**
- Returns the round number for a slot: 0 for FF, 1-6 for R1-R6.
- Used by downstream code for grouping results by round.
