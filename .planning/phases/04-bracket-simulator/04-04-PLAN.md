1. Create `src/simulator/score_predictor.py` with:

**Constants (from research -- historical championship game analysis 2003-2025 excl. 2020):**
```python
HISTORICAL_MEAN_TOTAL = 140.0    # mean total points in championship games
HISTORICAL_MEAN_TEMPO = 67.0     # fallback adj_t when team data unavailable
TEMPO_COEF = 3.43                # linear coefficient for avg_adj_t -> total
TEMPO_INTERCEPT = -89.7          # intercept for tempo -> total formula
TOTAL_MIN = 100                  # floor for predicted total
TOTAL_MAX = 180                  # ceiling for predicted total
```

**Function `predict_championship_score(team_a_id, team_b_id, win_prob_a, stats_lookup, season=2025)`:**
- `team_a_id`: kaggle_team_id of the predicted winner (or higher-probability team).
- `team_b_id`: kaggle_team_id of the predicted loser.
- `win_prob_a`: probability that team_a wins the championship (float in (0, 1)).
- `stats_lookup`: dict keyed by (season, kaggle_team_id) from `build_stats_lookup()`.
- `season`: tournament year.

Implementation:
1. Look up adj_t for both teams from stats_lookup. If a team is missing or adj_t is unavailable, use HISTORICAL_MEAN_TEMPO (67.0) as fallback. Log a warning if fallback is used.
2. Compute average tempo: `avg_tempo = (adj_t_a + adj_t_b) / 2.0`
3. Predict total points: `predicted_total = round(TEMPO_COEF * avg_tempo + TEMPO_INTERCEPT)`
4. Clamp total to [TOTAL_MIN, TOTAL_MAX].
5. Predict margin from win probability:
   - `predicted_margin = round((win_prob_a - 0.5) * 20 + 8)`
   - This scales: win_prob 0.5 -> margin 8, win_prob 0.75 -> margin 13, win_prob 0.89 -> margin 16.
   - Clamp margin to >= 1 (winner must win by at least 1).
   - Clamp margin to < predicted_total (margin cannot exceed total).
6. Compute individual scores:
   - `winner_score = (predicted_total + predicted_margin) // 2`
   - `loser_score = predicted_total - winner_score`
   - If winner_score <= loser_score (rounding error), adjust: `winner_score = loser_score + 1`.
7. Return dict:
   ```python
   {
       'predicted_total': int(predicted_total),
       'predicted_margin': int(predicted_margin),
       'winner_score': int(winner_score),
       'loser_score': int(loser_score),
       'winner_team_id': int(team_a_id),
       'loser_team_id': int(team_b_id),
   }
   ```

Add a `__main__` block that:
- Builds stats_lookup.

- Calls predict_championship_score with win_prob=0.65.
- Prints the predicted score.
- Verifies: total is plausible (100-180), margin >= 1, winner_score > loser_score.
