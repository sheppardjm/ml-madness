**Function `simulate_bracket(seedings, predict_fn, mode='deterministic', n_runs=10000, seed=None, override_map=None, slots_csv=None, season=2025, stats_lookup=None)`:**
- This is the main entry point for all simulation modes. For this plan, implement ONLY the `mode='deterministic'` path. For `mode='monte_carlo'`, raise `NotImplementedError("Monte Carlo mode not yet implemented -- see 04-03")`.
- Parameters:
  - `seedings: dict[str, int]` -- seed_label -> kaggle_team_id (from load_seedings)
  - `predict_fn` -- callable(team_a_id, team_b_id) -> float (team_a has lower seed)
  - `mode: str` -- 'deterministic' or 'monte_carlo'
  - `n_runs: int` -- Monte Carlo runs (ignored for deterministic)
  - `seed: int | None` -- random seed (ignored for deterministic)
  - `override_map: dict[str, int] | None` -- slot_id -> team_id overrides (ignored in this plan; 04-05 adds support)
  - `slots_csv: str | None` -- path to slots CSV (defaults to bracket_schema.SLOTS_CSV)
  - `season: int` -- year to load slot tree for
  - `stats_lookup: dict | None` -- stats dict for score prediction (used by 04-04)

**Deterministic implementation:**
1. Build slot tree via `build_slot_tree(season, slots_csv or SLOTS_CSV)`.
2. Build team-seed map via `build_team_seed_map(seedings)`.
3. Initialize `slot_winner: dict[str, int]` (slot_id -> kaggle_team_id) and `slot_prob: dict[str, float]` (slot_id -> win probability of the winner).
4. Traverse slots in topological order from `slot_tree['order']`:
   - For each slot, resolve StrongSeed and WeakSeed to team IDs:
     - If the reference is a seed label (exists in seedings dict), use `seedings[ref]`.
     - Otherwise it's a prior slot ID -- use `slot_winner[ref]`.
     - If either team is None, raise `ValueError` with descriptive message.
   - Enforce canonical seed ordering for predict_fn:
     - Look up `seed_a = team_seed_map[team_a]` and `seed_b = team_seed_map[team_b]`.
     - If `seed_a <= seed_b`: `prob_a_wins = predict_fn(team_a, team_b)`.
     - Else: `prob_a_wins = 1.0 - predict_fn(team_b, team_a)`.
     - Here `team_a` is the StrongSeed team and `team_b` is the WeakSeed team.
   - Pick the winner: if `prob_a_wins >= 0.5`, winner is team_a with prob prob_a_wins; else winner is team_b with prob `1.0 - prob_a_wins`.
   - Store in slot_winner and slot_prob.
5. Build the output dict:
   ```python
   from src.simulator.bracket_schema import ROUND_NAMES, slot_round_number

   result = {
       'mode': 'deterministic',
       'season': season,
       'slots': {},
       'champion': {
           'team_id': slot_winner['R6CH'],
           'win_prob': slot_prob['R6CH'],
       },
   }

   for slot_id in slot_tree['order']:
       round_num = slot_round_number(slot_id)
       result['slots'][slot_id] = {
           'team_id': int(slot_winner[slot_id]),
           'win_prob': float(slot_prob[slot_id]),
           'round': ROUND_NAMES.get(round_num, f'Round {round_num}'),
       }
   ```
6. Return the result dict. All values must be native Python types (int, float, str) for JSON serialization.
