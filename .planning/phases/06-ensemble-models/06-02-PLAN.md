**Function `run_optuna_sweep_lgb(df, n_trials=50) -> dict`:**
- Silence Optuna per-trial output: `optuna.logging.set_verbosity(optuna.logging.WARNING)`
- Define objective function:
  - Hyperparameter search space (from research Pattern 4):
    - `num_leaves`: int [10, 60] -- constrained to prevent overfitting on small dataset
    - `n_estimators`: int [50, 300]
    - `learning_rate`: float [0.01, 0.3] log-uniform
    - `min_child_samples`: int [10, 30] -- min 10 to prevent overfitting
    - `subsample`: float [0.6, 1.0]
    - `colsample_bytree`: float [0.5, 1.0]
    - `reg_alpha`: float [1e-4, 1.0] log-uniform
    - `reg_lambda`: float [1e-4, 10.0] log-uniform
  - For each fold from `walk_forward_splits(df)`:
    - Fit StandardScaler on training data ONLY
    - Build `LGBMClassifier(**params, class_weight='balanced', objective='binary', random_state=42, n_jobs=1, verbose=-1)`
    - Fit on scaled training data
    - Predict probabilities on scaled test data
    - Compute Brier score
  - Return `float(np.mean(fold_brier_scores))`
- Create study with `direction='minimize'`
- Run `study.optimize(objective, n_trials=n_trials)`
- Print best params and best Brier score
- Return `study.best_params`

**Function `evaluate_lgb(df, params) -> list[dict]`:**
- For each fold from `walk_forward_splits(df)`:
  - Fit StandardScaler on training data ONLY
  - Build LGBMClassifier with given params + `class_weight='balanced'`, `objective='binary'`, `random_state=42`, `n_jobs=1`, `verbose=-1`
  - Fit and predict
  - Compute and log Brier score AND log-loss per year
- Print per-year table with Brier and log-loss
- Print mean Brier and compare to baseline (0.1900)
- Return list of per-year result dicts: `[{'year': int, 'brier': float, 'log_loss': float, 'n_games': int}]`

**`__main__` block:**
1. Build matchup dataset: `df = build_matchup_dataset()`
2. Run Optuna sweep: `best_params = run_optuna_sweep_lgb(df, n_trials=50)`
3. Save params to JSON: write to `models/lgb_params.json` with `json.dump(..., indent=2)`
4. Evaluate with best params: `evaluate_lgb(df, best_params)`
5. Print comparison: "LightGBM mean Brier: X.XXXX vs baseline 0.1900"

Run: `uv run python -m src.models.train_lightgbm`

IMPORTANT NOTES:
- Use `class_weight='balanced'` (LightGBM supports this, unlike XGBoost)
- Use `verbose=-1` to suppress per-iteration output (NOT `verbosity=0` -- that's XGBoost's parameter)
- Set `n_jobs=1` to avoid thread contention during Optuna parallel trials
- Constrain `num_leaves` to max 60 to prevent overfitting on this ~1000-sample dataset
- `min_child_samples` >= 10 to prevent memorization of small training folds
- `random_state=42` for reproducibility
