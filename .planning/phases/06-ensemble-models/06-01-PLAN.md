**Imports:**
```python
from __future__ import annotations
import json

from xgboost import XGBClassifier

```

**Function `run_optuna_sweep_xgb(df, n_trials=50) -> dict`:**
- Silence Optuna per-trial output: `optuna.logging.set_verbosity(optuna.logging.WARNING)`
- Define objective function:
  - Hyperparameter search space (from research Pattern 3):
    - `n_estimators`: int [50, 300]
    - `max_depth`: int [2, 6]
    - `learning_rate`: float [0.01, 0.3] log-uniform
    - `subsample`: float [0.6, 1.0]
    - `colsample_bytree`: float [0.5, 1.0]
    - `min_child_weight`: int [1, 10]
    - `reg_alpha`: float [1e-4, 1.0] log-uniform
    - `reg_lambda`: float [1.0, 10.0] log-uniform
  - For each fold from `walk_forward_splits(df)`:
    - Fit StandardScaler on training data ONLY
    - Compute `scale_pos_weight = float((y_train == 0).sum() / (y_train == 1).sum())`
    - Build `XGBClassifier(**params, scale_pos_weight=scale_pos_weight, objective='binary:logistic', random_state=42, n_jobs=1, verbosity=0)`
    - Fit on scaled training data
    - Predict probabilities on scaled test data
    - Compute Brier score
  - Return `float(np.mean(fold_brier_scores))`
- Create study with `direction='minimize'`
- Run `study.optimize(objective, n_trials=n_trials)`
- Print best params and best Brier score
- Return `study.best_params`

**Function `evaluate_xgb(df, params) -> list[dict]`:**
- For each fold from `walk_forward_splits(df)`:
  - Fit StandardScaler on training data ONLY
  - Compute scale_pos_weight per fold
  - Build XGBClassifier with given params
  - Fit and predict
  - Compute and log Brier score AND log-loss per year
- Print per-year table with Brier and log-loss
- Print mean Brier and compare to baseline (0.1900)
- Return list of per-year result dicts: `[{'year': int, 'brier': float, 'log_loss': float, 'n_games': int}]`

**`__main__` block:**
1. Build matchup dataset: `df = build_matchup_dataset()`
2. Run Optuna sweep: `best_params = run_optuna_sweep_xgb(df, n_trials=50)`
3. Save params to JSON: `json.dump(best_params, open('models/xgb_params.json', 'w'), indent=2)`
4. Evaluate with best params: `evaluate_xgb(df, best_params)`
5. Print comparison: "XGBoost mean Brier: X.XXXX vs baseline 0.1900"

Run: `uv run python -m src.models.train_xgboost`

IMPORTANT NOTES:
- Do NOT use `class_weight='balanced'` with XGBClassifier -- use `scale_pos_weight` instead (XGBoost does not support class_weight)
- Set `verbosity=0` (not `verbose=-1` -- that's LightGBM's parameter)
- Set `n_jobs=1` to avoid thread contention during Optuna parallel trials
- `random_state=42` for reproducibility
