```python
def load_comparison_data() -> dict[str, dict]:
    """Load backtest results for baseline and ensemble models.

    Returns dict with model names as keys and their full result dicts as values.
    Files are read relative to project root (cwd).
    """
    baseline = json.loads(pathlib.Path("backtest/results.json").read_text())
    ensemble = json.loads(pathlib.Path("backtest/ensemble_results.json").read_text())
    return {"baseline": baseline, "ensemble": ensemble}
```

4. **`_compute_mean_upset_rate(data: dict) -> float`:**
Helper to compute mean upset detection rate from per_year entries.

```python
def _compute_mean_upset_rate(data: dict) -> float:
    """Compute mean upset detection rate across all backtest years."""
    rates = [y["upset_detection_rate"] for y in data["per_year"]]
    return sum(rates) / len(rates) if rates else float("nan")
```

5. **`_compute_mean_per_round_accuracy(data: dict) -> dict[str, float]`:**
Compute 4-year average per-round accuracy.

```python
def _compute_mean_per_round_accuracy(data: dict) -> dict[str, float]:
    """Compute 4-year average accuracy for each tournament round."""
    n_years = len(data["per_year"])
    result = {}
    for r in ROUND_NAMES:
        total = sum(y["per_round_accuracy"].get(r, 0.0) for y in data["per_year"])
        result[r] = total / n_years if n_years > 0 else 0.0
    return result
```
