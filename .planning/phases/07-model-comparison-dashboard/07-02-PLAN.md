**Function 1: `plot_per_round_accuracy(models_data, save_path="models/per_round_comparison.png")`**

Creates a grouped bar chart showing 4-year average per-round accuracy for baseline vs ensemble.

Implementation:
1. Define `rounds = ["Round of 64", "Round of 32", "Sweet 16", "Elite 8", "Final Four", "Championship"]`
2. For each model (baseline, ensemble), compute the 4-year average accuracy for each round:
   ```python
   def _avg_round_accuracy(data: dict, round_name: str) -> float:
       years = data["per_year"]
       return sum(y["per_round_accuracy"].get(round_name, 0.0) for y in years) / len(years)
   ```
3. Create grouped bar chart:
   - x-axis: 6 round names (use short labels: "R64", "R32", "S16", "E8", "FF", "CH")
   - Two bars per group: baseline (steelblue), ensemble (darkorange)
   - Bar width = 0.35
   - y-axis: Accuracy (0.0 to 1.0)
   - Title: "Per-Round Accuracy: Baseline vs. Ensemble (2022-2025 avg)"
   - Legend: "Baseline (LR)", "Ensemble"
   - Add value labels on top of each bar (f"{val:.0%}")
   - figsize=(10, 6)
4. `plt.tight_layout()`, `plt.savefig(save_path, dpi=150, bbox_inches="tight")`, `plt.close()`
5. Print confirmation: `f"Per-round comparison chart saved to: {save_path}"`

**Function 2: `plot_brier_heatmap(models_data, save_path="models/brier_heatmap.png")`**

Creates a heatmap with rows=models, columns=years, cells=Brier score.

Implementation:
1. `years = [2022, 2023, 2024, 2025]`
2. `model_names = list(models_data.keys())` — this will be ["baseline", "ensemble"]
3. Build matrix: for each model, extract per-year Brier scores:
   ```python
   matrix = []
   for name in model_names:
       row = []
       for yr in years:
           year_entry = next((y for y in models_data[name]["per_year"] if y["year"] == yr), None)
           row.append(year_entry["brier"] if year_entry else float("nan"))
       matrix.append(row)
   ```
4. Create heatmap using `ax.imshow()`:
   - cmap="RdYlGn_r" (red=high Brier=bad, green=low Brier=good)
   - aspect="auto"
   - vmin=0.12, vmax=0.25 (covers the range of observed Brier scores)
   - x-tick labels: years
   - y-tick labels: model names
   - Title: "Brier Score by Model and Year (lower is better)"
5. Annotate each cell with the Brier value: `ax.text(j, i, f"{val:.4f}", ha="center", va="center", fontsize=10)`
6. Add colorbar with label "Brier Score"
7. figsize=(8, 3) — only 2 rows so keep it compact
8. `plt.tight_layout()`, `plt.savefig(save_path, dpi=150, bbox_inches="tight")`, `plt.close()`
9. Print confirmation: `f"Brier heatmap saved to: {save_path}"`

IMPORTANT NOTES:
- Always call `matplotlib.use("Agg")` BEFORE `import matplotlib.pyplot as plt`
- Always call `plt.close()` after `plt.savefig()` to free memory
- Use `pathlib.Path(save_path).parent.mkdir(parents=True, exist_ok=True)` before savefig to ensure directory exists (models/ should exist but be safe)
- Only plot baseline and ensemble (not XGB/LGB — they don't have per-round or per-year bracket-simulation data)
