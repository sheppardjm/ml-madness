BACKTEST_YEARS = [2022, 2023, 2024, 2025]

**Function `walk_forward_splits(df, backtest_years=None)`:**
- Takes a matchup DataFrame (from `build_matchup_dataset()`) with a `Season` column.
- Default `backtest_years` is `BACKTEST_YEARS`.
- For each `test_year` in `backtest_years`:
  - `train_df` = rows where `Season < test_year`
  - `test_df` = rows where `Season == test_year`
  - Assert `len(train_df) > 0` with message: f"No training data before {test_year}"
  - Assert `len(test_df) > 0` with message: f"No test data for {test_year}"
  - **DATA LEAKAGE GUARD**: Assert `train_df['Season'].max() < test_year` with message: "DATA LEAKAGE: future year in training fold"
  - Yield tuple: `(test_year, train_df, test_df)`
- This is a generator function (uses yield).

**Function `describe_splits(df, backtest_years=None)`:**
- Calls `walk_forward_splits()` and prints a summary table:
  - Year | Train Seasons | Train Games | Test Games | Train Label=1 Rate
- Useful for sanity-checking the splits before training.

Add a `__main__` block that:

3. Calls `describe_splits()` to print the split summary
4. Runs a quick leakage check: for each fold, prints the max training season and confirms it's < test_year

Create `src/models/train_logistic.py` with:

**Imports:** sklearn.linear_model.LogisticRegression, sklearn.preprocessing.StandardScaler, sklearn.metrics.brier_score_loss, optuna, joblib, sklearn, pathlib, numpy

**Function `run_optuna_sweep(df, n_trials=50)`:**
- Builds walk-forward folds using `walk_forward_splits(df)`.
- Defines an Optuna objective function:
  - `C = trial.suggest_float("C", 1e-3, 100.0, log=True)`
  - For each fold: fit StandardScaler on training data only, transform both train and test, fit LogisticRegression(C=C, class_weight='balanced', solver='lbfgs', max_iter=1000, random_state=42), predict probabilities on test set.
  - Return mean Brier score across all 4 folds (minimize).
- Creates an Optuna study with `direction="minimize"` and `optuna.logging.set_verbosity(optuna.logging.WARNING)` to suppress per-trial output.
- Runs `study.optimize(objective, n_trials=n_trials)`.
- Prints best C and best Brier score.
- Returns `study.best_params['C']`.

**Function `train_and_save(df, best_C, model_path='models/logistic_baseline.joblib')`:**
- Extracts X (FEATURE_COLS) and y (label) from the full dataset.
- Fits StandardScaler on full X.
- Transforms X.
- Fits LogisticRegression(C=best_C, class_weight='balanced', solver='lbfgs', max_iter=1000, random_state=42) on full (scaled) X, y.
- Creates the `models/` directory if it does not exist: `pathlib.Path(model_path).parent.mkdir(parents=True, exist_ok=True)`
- Saves artifact dict via joblib.dump:
  ```python
  artifact = {
      'model': clf,
      'scaler': scaler,
      'feature_names': FEATURE_COLS,
      'train_seasons': sorted(df['Season'].unique().tolist()),
      'best_C': best_C,
      'sklearn_version': sklearn.__version__,
  }
  joblib.dump(artifact, model_path)
  ```
- Prints: model path, number of training games, coefficient signs, intercept.
- Returns the artifact dict.

**Function `load_model(model_path='models/logistic_baseline.joblib')`:**
- Loads artifact via `joblib.load(model_path)`.
- Prints sklearn version check (warning if different).
- Returns tuple: `(model, scaler, feature_names)`.

**Function `predict_matchup(team_a_features, model, scaler, feature_names)`:**
- Takes a dict of features (keys matching FEATURE_COLS), the model, scaler, and feature_names.
- Converts to numpy array in correct feature order, reshapes to (1, n_features).
- Scales with scaler.transform().
- Returns `model.predict_proba(X_scaled)[0, 1]` — probability that team_a wins (label=1).

Add a `__main__` block that:
1. Imports `build_matchup_dataset` from `src.models.features`.

4. Trains on full data with best C and saves to `models/logistic_baseline.joblib`.
5. Loads the model back and verifies it can predict:
   - Pick one row from the dataset, extract features, call `predict_matchup()`.
   - Print the prediction probability.
   - Verify it's between 0.05 and 0.95 (sanity check — not extreme).
6. Run: `uv run python -m src.models.train_logistic`
