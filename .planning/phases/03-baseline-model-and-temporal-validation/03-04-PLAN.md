2. **Modify `run_optuna_sweep()`** — In the objective function, after fitting `clf` on the training fold, wrap it with `CalibratedClassifierCV(clf, method='isotonic', cv='prefit')` and fit the calibrator on the SAME training fold data (this is acceptable because CalibratedClassifierCV with cv='prefit' just learns a monotonic mapping on the given data — the walk-forward split already ensures no test data leakage). Then use `calibrated_clf.predict_proba()` instead of `clf.predict_proba()` for the Brier score computation.

   Specifically, inside the `for _year, train_df, test_df in walk_forward_splits(df):` loop:
   ```python
   clf = LogisticRegression(C=C, class_weight="balanced", solver="lbfgs", max_iter=1000, random_state=42)
   clf.fit(X_train_scaled, y_train)

   # Post-hoc isotonic calibration on training data
   calibrated_clf = CalibratedClassifierCV(clf, method="isotonic", cv="prefit")
   calibrated_clf.fit(X_train_scaled, y_train)

   y_prob = calibrated_clf.predict_proba(X_test_scaled)[:, 1]
   ```

   IMPORTANT: The Optuna sweep must now select C using the CALIBRATED model's Brier score, so the optimal C accounts for the calibration step.
