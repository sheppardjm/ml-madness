    1. Create `src/ingest/parse_tourney.py` with two functions:

    **`ingest_tournament_games(raw_dir: str = "data/raw/kaggle", out_dir: str = "data/processed") -> int`**
    Uses DuckDB to:
    - Read MNCAATourneyCompactResults.csv, join with MTeams.csv (for team names) and MSeasons.csv (for DayZero to compute GameDate)
    - Filter to Season >= 2003 AND Season != 2020
    - Compute GameDate as `DayZero::DATE + DayNum` (DayZero comes from MSeasons.csv)
    - Add `IsFirstFour` boolean: True when DayNum IN (134, 135)
    - Add `Round` string column using DAYNUM_ROUND_MAP from src/utils/seasons.py. IMPORTANT: For DayNum=134 games in seasons BEFORE 2011, set Round to 'Play-In Game' instead of 'First Four' (the First Four format began in 2011; pre-2011 had a single play-in game)
    - Add `WLoc = 'N'` constant (all tournament games are neutral site -- WLoc column does NOT exist in tournament results CSV)
    - Order by Season, DayNum
    - Write to `{out_dir}/tournament_games.parquet` with zstd compression
    - Return the row count

    Output columns: Season, DayNum, GameDate, WTeamID, WTeamName, WScore, LTeamID, LTeamName, LScore, NumOT, WLoc, IsFirstFour, Round

    **`ingest_tournament_seeds(raw_dir: str = "data/raw/kaggle", out_dir: str = "data/processed") -> int`**
    Uses DuckDB to:
    - Read MNCAATourneySeeds.csv, join with MTeams.csv
    - Parse the Seed column (format: `W01`, `X16a`, `Z11b`):
      - Region: first character (W, X, Y, Z)
      - SeedNum: next 2 digits as integer
      - IsFirstFour: True if seed ends with 'a' or 'b'
    - Filter to Season >= 2003 AND Season != 2020
    - Write to `{out_dir}/seeds.parquet` with zstd compression
    - Return row count

    Output columns: Season, TeamID, TeamName, Seed (original string), Region, SeedNum, IsFirstFour
