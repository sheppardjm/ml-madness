# Requirements: March Madness 2026 Bracket Predictor

**Defined:** 2026-03-02
**Core Value:** Accurate, data-driven bracket predictions that give a competitive edge in bracket challenges

## v1 Requirements

### Data Pipeline

- [x] **DATA-01**: Historical tournament data backfilled from Kaggle (20+ years of seasons) — 22 seasons (2003-2025 excl. 2020) in Parquet
- [x] **DATA-02**: Current 2025-26 season stats pulled from CBBpy + cbbdata APIs — 2024-25 as proxy via cbbdata archive
- [x] **DATA-03**: Team name normalization mapping across all data sources (ESPN, Kaggle, Sports-Reference, cbbdata) — 381 teams, 101 aliases
- [x] **DATA-04**: Auto-fetch 68-team bracket from ESPN unofficial API with manual CSV fallback — pipeline operational

### Modeling

- [x] **MODL-01**: Logistic regression baseline model trained on efficiency differentials — Brier=0.1900
- [x] **MODL-02**: XGBoost + LightGBM stacking ensemble producing calibrated win probabilities — TwoTierEnsemble, Brier=0.1692 (-11%)
- [x] **MODL-03**: Walk-forward temporal validation (train on years 1-N, test on year N+1) — 4-fold temporal CV
- [x] **MODL-04**: Model evaluation using Brier score and log-loss (not just accuracy) — per-year metrics logged

### Simulation

- [x] **SIML-01**: Monte Carlo bracket simulation (10K+ runs) propagating probabilities through all 67 games — vectorized numpy, 0.21s for 10K runs
- [x] **SIML-02**: Champion prediction with confidence percentage — 31.8% for 2025 test
- [x] **SIML-03**: Predicted championship game score — tempo-based formula (exists in simulator; not wired to UI)
- [x] **SIML-04**: Round-by-round advancement probabilities for each team — 7 round milestones

### Backtesting

- [x] **BACK-01**: Backtest model against 2025 tournament using in-repo results data — temporal isolation verified
- [x] **BACK-02**: Multi-year holdout validation across 2022-2025 (chalk + upset-heavy years) — reproducible results
- [x] **BACK-03**: Model comparison dashboard showing baseline vs ensemble performance — CLI table + charts + selected.json

### Web UI

- [x] **WEBU-01**: Visual 68-team bracket display with predicted winners — SVG bracket in Streamlit
- [x] **WEBU-02**: Win probabilities shown per game matchup — rendered in each slot
- [x] **WEBU-03**: Interactive pick overrides with downstream cascade recalculation — session persistence + reset
